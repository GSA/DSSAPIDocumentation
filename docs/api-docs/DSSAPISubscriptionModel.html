Below is the description of the subscription model and step-by-step instructions on how to subscribe to DSS APIs.

Step 1:Register & Acquire API Key

Goto 'https://api.data.gov'

Click on 'Get an API Key'

Fill in the necessary information and click ‘Signup’.

Note: It is recommended that you use a group email address if you are intending to use the APIs from within an application / business process.

You will get to the following screen:

You will receive an email from api.data.gov on the email address provided in the request. The email will include an API Key that will uniquely identify you as the API subscriber across all the APIs available via api.data.gov

Step 2: Forward the email you received from api.data.gov to dssteam@gsa.gov with the Subject line as ‘New subscriber to DSS API’

Step 3 & 4: DSS Team will configure the API key you provided as a valid subscriber of the DSS APIs and you will receive an approval email from dssteam@gsa.gov notifying successful completion of the API configuration.

The email will also include the API endpoints

Step 5 & 6: You can now start invoking the available DSS APIs by providing the API Key with each API call.

E.g.

https://api.gsa.gov/sandbox/operations/signature/Packages/Create?api_key=PjksItI6iqfW7WfkiOAIpwGrPvnKSOtsdDBDEqs6

Step 6: Please forward the email you receive from data.gov to the DSS Team at the following email address: ‘dssteam@gsa.gov’ with the Subject line as ‘New subscriber to DSS API’ and provide your email address that was provided on the ‘Signup’ screen

Step 7: The DSS Team will add your api key as the valid subscriber to the DSS APIs. You can then start using the DSS APIs. Additional information about the APIs is available at https://gsa.github.io/DSSAPIDocumentation/api-docs/
